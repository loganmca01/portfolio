Customizable Toy Assembly Interpreter (in progress)
Github Repo

- Toy assembly interpreter where user can enter an instruction set using a custom, easy-to-read register transfer notation, then run the commands dictated in the instruction set in a virtual environment.
- Designed for use in Computer Organization class to introduce assembly concepts without full assembly syntax.
- Two main components:

1. Instruction set parser: Takes a file input of instructions written in register transfer notation. Builds abstract syntax trees for each command, then puts them into symbol table for use in the user environment.
2. User command interpreter: Takes user commands (file or stdin) and runs them in virtual environment with an array of chars representing the stack, and variables for registers.
